# brand/

Canonical sources for the justnotes visual identity. Everything else
(Tauri icons, favicons, OG image, marketing illustrations) derives from
these — re-generate when the source changes.

```
brand/
├── mark.svg          canonical symbol (amber dot + halo on canvas bg)
├── wordmark.svg      mark + "justnotes" set in Geist Medium
├── mark-1024.png     raster master for `tauri icon`
├── canvas-hero.svg   marketing landing-page hero illustration
└── og.png            1200×630 social card (PIL-rendered)
```

Colors come from the active JustUI theme when the SVG is inlined into
HTML — `currentColor` and `var(--accent)` resolve against the consuming
document. Standalone renders (favicon, OG) fall back to the espresso-dark
hex values declared inside each SVG's `<style>` block.

## Re-generating raster derivatives

PNGs and the favicon ICO are generated by `brand/build.py` (Python +
PIL). Run from the repo root:

```sh
python3 brand/build.py
```

The script writes:

- `src-tauri/icons/{32x32,128x128,128x128@2x,icon}.png` (replaces placeholders)
- `apps/web/public/favicon.{svg,ico}` + `og.png`
- `apps/marketing/public/favicon.{svg,ico}` + `og.png`

For the Tauri .icns/.ico set:

```sh
pnpm --filter root exec tauri icon brand/mark-1024.png
```

That overwrites the PNGs `build.py` wrote and also produces the
platform-specific bundles.
